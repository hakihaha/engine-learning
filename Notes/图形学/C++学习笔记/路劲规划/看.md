# 一年引擎开发成长路线图（52 周 · BBB 版本）

> 目标：从 0 → 具备初级引擎 / 图形学工程师能力  
> 学习强度：每天 2 小时（≈14h / 周）  
> 侧重方向：C++ / 图形学 / 渲染 / 引擎基础  
> 原则：每一阶段都必须留下【能跑的代码】

---

## 阶段 0：准备阶段（第 0 周）
**目标：搭好一整年的开发环境**

- 安装 VS2022（C++ 桌面开发）
- 安装 CMake
- 安装 Git + GitHub 账号
- 准备一个专用学习仓库：`engine-learning`
- 建立笔记目录：`/notes`

✅ 产出：
- 空的 CMake 项目能成功编译
- Git 仓库初始化完成

---
