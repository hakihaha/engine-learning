# 学习路径

## 目标
- 夯实 C++ 基础 → 熟到能看懂/写出引擎层代码
- 掌握现代渲染基础 GPU/Shader → 至少能做简单渲染器
- 掌握引擎架构与模块（资源、事件、组件、渲染循环）
- 至少能跟着课程做出一个小型可运行的引擎 Demo
- 阅读部分开源引擎源代码（HLE）

## C++
- 侯捷 C++ 系列
    - C++面向对象
    - 模板与 STL
    - 内存管理（也就是 GC、allocator）
- **清华 Chen Hao：现代 C++ 课程（C++11/14/17）**
- **Udemy: Tim Buchalka’s C++ Masterclass**（入门~中级）
- **“C++ Nanodegree”（Udacity）** —— 工程化很强
- **CPPCon 会议视频（YouTube）** —— 现代 C++ 必修

## 图形学➕渲染基础
- 图形学 / 渲染管线基础课程（Battlefire）
    - OpenGL / GLSL（Battlefire）
    - 顶点着色器、片元着色器
    - 光照、阴影、贴图
    - Framebuffer / RenderTexture
    - OpenGL 渲染循环
- **LearnOpenGL (全网最强)** [https://learnopengl.com](https://learnopengl.com)
- **GAMES101（清华）**

## 现代渲染器实现
- **OpenGL 渲染器实战项目**
- **Vulkan 入门 & 实战项目**
